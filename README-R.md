## Установка и сборка:

Для установки, компиляции и запуска без интерфейса введите:
```cpp
git clone https://github.com/BulgakovDmitry/3D_triangles
cd 3D_triangles
cmake -S . -B build
cd build
cmake --build .
./3D_triangles 
```

Если хотите получить дебажную сборку введите:
```cpp
cmake -S . -B build -DCMAKE_BUILD_TYPE=Debug
cd build
cmake --build .
./3D_triangles
```
